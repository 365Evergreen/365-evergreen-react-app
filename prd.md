This project is a website refresh for 365 Evergreen, using Azure Static Web App for a frontend and WordPress for content.

This site will initially have no auth requirements

The site will only act as a frontend to serve WP content

## Tech stack
1. React/Typescript
2. Azure SWA
3. WOrdPress
    - GraphQL
    - ACF
4. Azure Blob Storage
    - anonymous access (already in place)
